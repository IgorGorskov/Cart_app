Это учебный проект, предназначенный для практики разработки веб-приложений с базовой функциональностью для работы с продуктовой корзиной и списком желаемого. Приложение предоставляет пользователям возможность регистрироваться, авторизоваться, добавлять и удалять товары в корзину, а также управлять списком желаемого
Этот проект создан исключительно в учебных целях для демонстрации и отработки навыков работы с пользовательской аутентификацией и базовыми операциями управления товарами.

Backend: Express.js
Frontend: HTML/CSS/React.js
